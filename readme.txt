== Installation ==

This section describes how to install the plugin and get it working.

1. Download the zipped file of the plugin
1. Upload the MacDock folder to your WordPress Blog's plugin directory
1. Activate the plugin from you blogs Admin panel
1. Customize the plugin from Settings>MacDock page.
